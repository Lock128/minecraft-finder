## Detailed Feature Description (for long description updates)

Edition & Version Support
- Java Edition or Bedrock Edition — the app now uses the correct random number generator for each edition, so ore predictions match your actual world
- Pre-1.18 (Legacy) mode uses the classic uniform ore distribution with fixed Y ranges: diamonds at Y 1–15, gold at Y 1–31, iron at Y 1–63, and more
- 1.18+ (Modern) mode uses the current triangular distribution with the expanded -64 to 320 world depth
- Bedrock Edition uses a Mersenne Twister RNG to replicate Bedrock's C++ engine
- Contextual info boxes let you know when predictions are approximate (Bedrock) or when legacy sweet spots apply
- Selections are remembered between sessions — no need to re-pick every time
